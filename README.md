# ⚡ SalesForge AI — Forecasting Engine

An AI-powered sales forecasting and business analytics dashboard built using Streamlit, Prophet, Pandas, Matplotlib, and Seaborn.  
This project provides real-time insights, interactive visualizations, KPI dashboards, and future sales forecasting using machine learning. :contentReference[oaicite:0]{index=0}

---

## 🚀 Features

- 📊 Interactive Sales Dashboard
- 📈 AI-Powered Sales Forecasting using Prophet
- 📉 Deep Analytics & Trend Analysis
- 🧠 Business Intelligence Insights
- 🌍 Regional & Category-wise Performance
- 📅 Monthly & Quarterly Revenue Analysis
- 📦 Strategic Recommendations
- 📥 Forecast CSV Download
- 🎨 Modern Glassmorphism UI
- ⚡ Fully Responsive Streamlit App
